#######################################################################################
              OsmAnd Point of Interest converter to FreeMind
#######################################################################################

This utility converts map files from OsmAnd, which can be downloaded here:

  http://code.google.com/p/osmand/downloads/list?num=700

to a FreeMind mindmap.

All points of interest contained in the OsmAnd file will be sorted by type and subtype and its name.

Usage:

Linux: 

./OsmAndToFreeMindImporter.sh Germany_hessen_europe_1.obf Germany_hessen_europe_1.mm

Windows:

OsmAndToFreeMindImporter.bat Germany_hessen_europe_1.obf Germany_hessen_europe_1.mm
